# Ashok Leyland Accident Analytics Dashboard

## Overview

Developed a Power BI dashboard during my internship at Ashok Leyland to analyse accident trends and transporter performance across logistics operations.
Note: Numbers have been tampered with, in order to not reveal any confidential information of the company.

## Scope

- Analysed 205 accident records
- Built 40+ KPIs
- Created interactive visualisations for operational decision-making
- Identified accident hotspots and transporter performance patterns

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dashboard Highlights

- Accident timing analysis
- Route-wise accident tracking
- Transporter performance analysis
- Vehicle classification analysis
- Distance travelled before accident
- Geographic accident mapping

## Key Outcome

Provided operational visibility into accident patterns and transporter accountability, supporting data-driven decision making within outbound logistics operations.
